# 王子墨人物与资产总规范

## 管理范围

本页统一管理人物身份、声音、服装、表情和场景资产。当前资产尚未录入时，不得凭空宣称已有固定锚点或授权素材。

## 使用规则

- 每个项目开始时登记本次选用的身份、品牌、场景、声音与道具来源。
- 同一项目内冻结关键参考，不在镜头间随意更换。
- 参考图只按登记用途使用，状态参考不得越权替代身份或品牌主锚点。
- 新资产进入 `attachments/`，原始项目副本进入 `08_原始项目副本/`。
- 不保存 API Key、密码、会话令牌或不必要的敏感信息。
