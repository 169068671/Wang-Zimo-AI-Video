# 视频成片审核清单

- [ ] 身份、品牌与场景资产符合本项目登记
- [ ] 人物脸型、肤质、手部、肢体和服装稳定
- [ ] 商品、道具、文字、Logo 和空间结构正确
- [ ] 镜头方向、动作承接、光线、色彩和时间连续
- [ ] 配音、口型、音乐、音效与字幕同步
- [ ] 无明显闪烁、变形、穿模、跳帧或物理错误
- [ ] 分辨率、比例、时长、码率和平台要求正确
- [ ] 已保存母版、发布版、提示词、任务 ID 和成本记录
