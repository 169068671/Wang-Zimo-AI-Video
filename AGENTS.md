# 王子墨 AI 视频仓库使用指令

## 任务开始

1. 先读取 [[00_HOME/启动必读]]。
2. 再读取 [[01_人物资产/王子墨人物与资产总规范]]。
3. 按任务类型读取工作流、项目和模板，不扫描整个仓库。

## 硬规则

- 新项目先登记并冻结身份、品牌、场景、声音和道具资产。
- 原始高分辨率锚点只用于本地生成与验收；向 API / GitHub / 在线服务上传参考图时，必须使用 `attachments/上传锚点/` 对应小图，禁止上传大文件。详见 [[02_创作工作流/上传锚点与GitHub同步规则]]。
- 付费生成前实时查询模型、参数、余额和预计成本并取得确认。
- 异步任务只提交一次，保存任务 ID，只轮询原任务。
- 不保存 API Key、密码、会话令牌或不必要的敏感信息。
- 新增标签登记到 [[00_HOME/标签索引]]；新增核心笔记登记到 [[00_HOME/内容索引]]。

## 任务结束

- 更新项目、任务和成本记录，完成成片审核。
- 运行 `python3 99_System/validate_vault.py`，通过后才能宣布知识库更新完成。
