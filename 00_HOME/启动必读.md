# 启动必读

1. 先读 [[首页]]、[[内容索引]] 和 [[标签索引]]。
2. 开始制作前读取 [[../01_人物资产/王子墨人物与资产总规范]]。
3. 按 [[../02_创作工作流/AI视频标准制作流程]] 执行，并在 [[../04_项目/项目总览]] 登记项目。
4. 付费生成前确认模型、参数、预计成本；异步任务只提交一次并保存任务 ID。
5. 完成后执行 [[../05_审核/视频成片审核清单]]，再运行知识库核验。
